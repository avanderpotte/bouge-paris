## Technologies
- AngularJS 1
- Chartist
- SASS
- Bower

## Setup

``` bash
# install dependencies
npm i -g gulp bower
npm i

# serve with hot reload at localhost:3000
gulp watch

# build for production with minification
gulp build
```
